# Weather Dashboard
A full stack weather_dashboard application built with A full-stack Weather Dashboard built with **Node.js**, **Express.js**, **MongoDB Atlas**, and the **Open-Meteo API**. In this dashboard we can enter any city and search for its weather and forecast data.

---
## 📌 Features

- 🔍 Search weather by city name
- 🌍 Automatic city geocoding
- 🌡️ Current weather information
- 📅 Multi-day weather forecast
- 💾 Stores search history in MongoDB Atlas
- 📜 Displays recent search history
- ⚡ Fast and responsive user interface
- ❌ Error handling for invalid city names and API failures

---
## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (ES6)

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- MongoDB Node.js Driver

### APIs
- Open-Meteo Weather API
- Open-Meteo Geocoding API
